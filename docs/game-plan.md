# 🔺 GeoQuest — 글로벌 교육용 수학 도형 게임 기획안

> 현재 "삼각형 찾기 놀이터"를 **글로벌 교육용 수학 도형 게임**으로 확장하기 위한 종합 기획안

---

## 서비스 콘셉트

**"도형 속에 숨은 보물을 찾아라!"**

단순히 삼각형을 찾는 퍼즐에서 출발하여, **다양한 도형 + 난이도 시스템 + 게이미피케이션**을 결합한 글로벌 교육 게임.  
경쟁작(DragonBox, Euclidea, Prodigy Math) 대비 **직관적인 터치 인터랙션**과 **아름다운 비주얼**이 핵심 차별점.

---

## 1. 난이도별 레벨 시스템 (월드맵 구조)

| 월드 | 테마 | 대상 연령 | 도형 | 핵심 개념 |
|------|------|-----------|------|-----------|
| **1. 삼각숲** | 숲 속 탐험 | 6–8세 | 삼각형 | 도형 인식, 꼭짓점 세기 |
| **2. 사각도시** | 미래 도시 | 8–10세 | 사각형·평행사변형 | 대각선, 넓이 개념 |
| **3. 다각별** | 우주 스테이션 | 10–12세 | 오각형·육각형 | 내각의 합, 대칭 |
| **4. 원과 부채꼴** | 해저 세계 | 12–14세 | 원·호·부채꼴 | 원주율, 호의 길이 |
| **5. 입체미궁** | 고대 유적 | 14세+ | 3D 전개도 추론 | 공간 지각력 |

### 각 월드 내 스테이지 구성
- **탐험 모드** (10스테이지): 기본 도형 찾기 → 복잡도 점진적 증가
- **챌린지 모드** (3스테이지): 시간 제한 + 제한된 힌트
- **보스 스테이지** (1스테이지): 복합 도형 퍼즐

---

## 2. 핵심 게임 기능

### 문제 유형
| 유형 | 설명 |
|------|------|
| **숨은 도형 찾기** | 복잡한 그림에서 특정 도형 찾기 (현재 방식 확장) |
| **도형 조합** | 작은 도형을 합쳐 큰 도형 만들기 |
| **도형 분해** | 큰 도형을 지정된 수로 분할 |
| **대칭 완성** | 반쪽 도형을 보고 나머지 완성 |
| **전개도 매칭** | 전개도와 3D 형태 연결 (고급) |

---

## 3. 게이미피케이션

- ⭐ **별점 시스템**: 각 스테이지 1–3개 별 (정확도 + 속도 + 힌트 미사용)
- 💎 **도형 크리스탈**: 게임 내 화폐 (스킨·커스터마이징 교환)
- 🏆 **업적 뱃지**: "삼각형 마스터", "100연속 정답" 등
- 📊 **글로벌 리더보드**: 주간/월간 랭킹 (국가별 필터)
- 👨‍🏫 **교실 모드**: 선생님이 학생에게 과제 할당 (B2B 확장)
